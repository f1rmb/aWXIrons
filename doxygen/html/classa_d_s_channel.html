void aDSChannel::setup ( uint8_t  pwmPin,
uint8_t  sensorPin,
uint8_t  ledPin 
)

Setup member function, should be called before any other member.

Pins will be embedded into aPin_t object, timer, mask, port and output register will be set also here, preventing using Arduino analog/digital{Read/Write}() calls, which are quite slow.

Parameters
pwmPinuint8_t : PWM pin, used to drive the output MosFET
sensorPinuint8_t : Sensor pin, used to get analog temperature value.
ledPinuint8_t : LED pin, used to reflect Heating/Cooling state
